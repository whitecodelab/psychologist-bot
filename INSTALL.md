Инструкция по установке бота психолога
Требования:
Python 3.8 или выше

Telegram аккаунт

Шаг 1: Установка файлов
Скачайте архив с файлами бота

Распакуйте в нужную папку

Откройте терминал в этой папке

Шаг 2: Установка зависимостей
Введите команду:

text
pip install -r requirements.txt
Шаг 3: Настройка бота
3.1 Получите токен бота:
Напишите @BotFather в Telegram

Отправьте команду: /newbot

Придумайте имя бота (например: "Psychologist Assistant")

Придумайте username бота (например: "my_psychologist_bot")

Скопируйте выданный токен (выглядит как: 1234567890:ABCdefGHIjklMNopQRstuVWXyz)

3.2 Узнайте свой ID:
Напишите @userinfobot в Telegram

Скопируйте ваш цифровой ID

3.3 Создайте файл .env:
В папке проекта создайте файл с именем .env и вставьте:

BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_Telegram_ID
DATABASE_URL=sqlite:///./psychologist_bot.db

ЗАМЕНИТЕ:

ваш_токен_от_BotFather на токен из шага 3.1

ваш_Telegram_ID на ID из шага 3.2

Шаг 4: Запуск бота
Введите команду:

text
python run.py
Проверка работы
Найдите вашего бота в Telegram по username

Напишите /start

Если вы администратор - увидите меню управления

Если обычный пользователь - увидите кнопку записи

Если возникли проблемы:
Проверьте что все шаги выполнены точно

Убедитесь что токен и ID вписаны правильно в .env

Убедитесь что Python установлен

Напишите разработчику для помощи

Бот готов к работе!
